# Vinary Viewer logo (final — "Full Automaton")

The mark: a folded-corner document whose outline is an inverted binary tree drawn
as a weighted pushdown automaton — gold start-state root, green arrowed transitions,
double-circle accepting states, and a token stream flowing below. Verdant palette,
shared DNA with the Vinary Tree, LLC shield (gold nodes, vine greens).

## Files
- `vinary-viewer.svg` — the mark alone, transparent, outlined (works on light & dark).
- `vinary-viewer-appicon.svg` — twilight-gradient rounded-square app icon (full detail).
- `vinary-viewer-favicon.svg` — simplified small-size mark on the same background.
- `png/appicon-*.png` — raster app icons. Detail tiers: full >=256, medium 128-48, simplified <=32.
- `png/favicon-32.png`, `png/favicon-16.png`, `favicon.ico` — favicons.

## Palette
- Deep green (transitions/outline): #3e6b26 - outline dark: #34531e
- Leaf green (fold, states): #6fa847 - light green (inner v): #8fbf62
- Gold (root, tokens): #d4a94a - Page: #f8f5ec - Text stream: #aebd9c
- App-icon background gradient: #46305e -> #2a1e3f -> #211737

## Usage
- GitHub avatar / README: appicon PNGs.
- Desktop icon: 1024/512/256 (macOS), favicon.ico (Windows), hicolor set (Linux).
- Wordmark lockup: mark + "Vinary Viewer" in Source Serif 4 Semibold, "Viewer" in #8fbf62.
